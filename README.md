# Smart Calculator for Android
A mobile calculator application developed in Java using Android Studio (Panda 3).

## Features
* Addition
* Subtraction
* Multiplication
* Division
* Calculation history storage using a database

## Technologies
* Java
* Android Studio
* SQL / Database

## Purpose
This project was created as a university learning project for practicing Android and database development.

## Future Improvements
- Add advanced mathematical operations
- Improve UI design
